# Context Saver Skill

> [English](README.md)

一个 Claude Code 技能（Skill），用于在长时间运行的实验/研究对话中保存和读取上下文快照，解决上下文窗口溢出问题。

## 功能

- **存档 (Save Archive)** — 将当前实验状态保存为快照（配置、结果、脚本、已知问题、下一步计划等）
- **读档 (Load Archive)** — 选择并恢复之前的存档，在新对话中继续工作
- **存档列表 (List Archives)** — 查看所有存档摘要
- **删档 (Delete Archive)** — 删除不需要的存档

## 安装

```bash
# 通过 npx 安装
npx skills add --path https://github.com/drw41823/claude-context-saver-skill
```

或者手动复制到 `~/.claude/skills/context-saver/`。

## 使用方法

在 Claude Code 中触发：

| 命令 | 用途 |
|------|------|
| `save-archive` / `存档` | 保存当前实验快照 |
| `load-archive` / `读档` | 载入存档 |
| `list-archives` / `存档列表` | 查看所有存档 |
| `delete-archive` / `删档` | 删除存档 |

所有存档保存在项目根目录的 `archives/` 文件夹中。

## 文件结构

```
context-saver/
├── SKILL.md       — Skill 主文件，包含完整流程和模板
├── README.md      — 英文说明
└── README.zh.md   — 中文说明
```

## 快照内容

每次存档包含最多 14 个部分：

1. **实验配置** — 模型类型、LoRA 参数、训练超参数、联邦学习设置、蒸馏配置
2. **环境配置** — conda 环境、GPU 信息、环境变量
3. **外部模型 & API 配置** — 使用的第三方模型、API 端点、密钥位置
4. **配置文件参数指南** — 常用可调参数说明
5. **实验结果汇总** — 所有实验的评测结果表格
6. **当前操作状态** — 正在做什么、最后执行的命令、完成状态
7. **错误与已知问题** — 每个 bug 的症状、根因、修复方法
8. **待办任务 / 下一步计划** — 带具体命令的优先级列表
9. **项目关键脚本索引** — 每个脚本的路径、用途、使用示例
10. **数据流文档** — 数据集路径、大小、格式、用途
11. **实验目录结构** — 命名规则、文件说明、结果文件位置
12. **持久化记忆** — CLAUDE.md 和 MEMORY.md 内容
13. **近期操作日志** — 按时间顺序的行动、命令、结果
14. **实验完整上下文** — 研究目标、当前阶段、已验证项、待验证假设

## 设计原则

- **完整性优先** — 宁可冗余，不可遗漏
- **可操作性** — 每个部分都能帮助新对话立即上手
- **精确命令** — 始终包含可直接执行的 shell 命令
- **不丢失未完成的工作** — 记录正在进行的工作的精确状态
- **键盘选择** — 所有存档选择使用方向键导航，无需手动输入文件名
